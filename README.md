# Latex Template

Use this template to write reports for university.

## Get Started

### Build

```bash
make
# or
tectonic -X build -k
```

### Watch

```bash
./watch.sh
# or
tectonic -X watch --exec "build -k"
```

## Standards

### Labels

| Type       | Prefix      |
|------------|-------------|
| Section    | sec:<name>  |
| Subsection | ssec:<name> |
| Image      | img:<name>  |
| Table      | tab:<name>  |
| Figure     | fig:<name>  |
| Algorithm  | alg:<name>  |
| Chapter    | ch:<name>   |
| Equation   | eq:<name>   |
| Listing    | lst:<name>  |
| Appendix   | apx:<name>  |
